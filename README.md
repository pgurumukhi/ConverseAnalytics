ConverseAnalytics
=================

## Required Setup -
1. Install python
2. Browsermob proxy


## 1) Python
  Install python 2.6 or 2.7.
## Python modules required - 
### Dominate
```sh
pip instsall dominate
```
### Browsermobproxy
```sh
pip install browsermob-proxy
```
or 
```sh
easy_install browsermob-proxy
```

## 2) Browsermob Proxy
   Install it from - http://bmp.lightbody.net/
